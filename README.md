# Engineering Case Studies

Selected technical case studies from systems I have designed, built, and operated as a full-stack engineer and technical co-founder. Focus areas: production AI features (function-calling agents, retrieval over domain data, ML-driven analysis), real-time data pipelines, and subscription products with multi-provider billing.

Each case study follows the same structure: context, constraints, decision rationale, implementation highlights, results, and what I would do differently.
